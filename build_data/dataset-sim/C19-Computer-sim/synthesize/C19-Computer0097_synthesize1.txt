1 设计原则与目标 
1.1 设计目标 
　　银川海关办公自动化网络系统采用计算机、通信、 网络和数据库等技术，建立起上与海关总署、下与各报关单位，以及与全国其它海关相连的系统，逐步实现处理自动化、数据传输网络化、重大决策科学化。它能对所使用的 3 Com 公司的网络设备提供端口管理，设置方便简单，排除故障容易，能对网络交换机、集线器、 路由器中每一端口进行管理和设置。 　　 Ananova把具有3D游戏风格的动画技术，和PA新媒体公司先进的即时新闻处理和发布系统结合到一起。 
　　采取有效措施，保证计算机网络系统可靠、安全地运行。 
3.2 网络管理软件 
　　网络管理平台使用 HP NetServer LH Pro 网络服务器随机带的 HP OpenView ，网络管理软件使用 3Com 公司的 Transcend Enterprise Manager Version 5.0 for Windows 。。。。 
　　交换机 3Com 公司的 Superstack   3000 TP 型，选用 Superstack ∏ HUB 100 型集线器，它具有 12 个 100M 接口。 它具有 Pentium Pro 200M CPU ， 64M 内存， 4G 热交换 Ultra SCSI 硬碟， 4 个 SATA 插槽， 4 个 EISA 插槽，一个 PCI/EISA 插槽；附有 3 个 Ultra SCSI 通道的 Plus NetRAID PCI SIMD控制器；内带一块 HP 10/100TX PCI 总线局域网接口卡；可选配 SureStore  磁带驱动器进行数据与系统的备份。 选用 3Com 公司的 Fast Etherlink XL PCI TX 网卡。这将使他们在跟踪和自己有关的信息时变得更为容易。 
3 MP1及软件平台 
3.1 网络操作系统 
　　选择了 Office 的“  XP ”作为网络管理软件。在这种方式下，银川海关计算机网络系统增加两台通信服务器， 远程报关单位也要一台通信机，都通过 modem 接入 PSTN 公共。海关由于其业务发展性质， 对网络的安全性给予了极大的关注，考虑到“ Windows NT ”在用户账号方面为应用系统提供了更高的灵活性和更好的帐户管理功能，在资源的安全性和审计方面也有很强的优势。 另外，对窗口软件的可视性与可操作性方面的习惯等因素，“ Windows NT ”成为首选。。即各报关单位与银川海关之间的数据通信是通过 PSTN 进行的。
