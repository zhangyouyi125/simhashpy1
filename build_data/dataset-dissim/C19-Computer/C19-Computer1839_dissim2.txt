运用Java Applet制作多媒体网页的方法和实例
傅献祯
摘 要： 用Java Applet插入HTML文档制作多媒体动态网页的一般方法，并给出制作实例。
关键词： Java语言 网页 Java Applet HTML语言 超级链接
1 Java Applet概述
　　众所周知，网页的制作可以利用超文本标识语言HTML去编写。用户可以各种层次将网络Cache放置在一些网络地区。这是由于代理服务器是所有用户的通道，而且，代理服务器需要配置每1个用户的浏览器，这对于企业级服务商是不可接受的。
　　利用HTML标识语言还可将Internet上连接的不同地区的服务器上的信息文件连接起来，控制不同超媒体文件之间的超级链接。
　　HTML通过标识符对文本的成分进行控制，用它建立的网页所包含的成分是有限的，仅有标题、段落、少量的列表和一些图形可供选择使用，因而HTML所提供的功能也是有限的，特别是在制作多媒体动态网页方面。
　　Java是一种面向对象的程序设计语言，它已经广泛地应用于Internet上，以完成HTML所无法完成的功能，具有简单高效、安全稳定、结构化等特点，由于它的结构化特性使得它易于传播。用户访问WWW的频度，是符合高斯分布的。基于Java对HTML的辅助性，通过超级链接方式点击某项链接后Java Applet即被启动，这将会使网页的画面更加绚丽多姿，使多媒体的优越性在网页中得以体现。
　　Cache技术是一种对于频繁访问Web信息的请求在本地实现的技术。制作时，首先要编写好Java小程序，然后把它嵌入到网页中。编制好Java Applet程序并编译成功后，就可把编译形成的．class文件导入到当前的站点中，这是制作动态网页的关键环节。在网页中插入Java Applet可借助FrontPage 98按如下的方法进行：
　　1．单击FrontPage 98编辑器中的“Insert”菜单，选中“Advanced”项，会出现1个子菜单，再选择菜单中的“Java Applet”项，FrontPage编辑器将打开“Java Applet properties”对话框。按我国目前Internet的使用情况，在1000台用户的园区网，若使用网络Cache，半年至1年就可收回成本。对于多数搜索引擎，从1天1次到2个月更新1次，平均是一星期更新1次。
　　Cache技术是一种对于频繁访问Web信息的请求在本地实现的技术。
　　4．有的Java浏览器可能不支持Java，需要显示1个提示信息，可以在对话框的“Message For Browsers Without Java Support”框内键入文本，Web浏览器将显示此文本以代替Java Applet。据统计目前我国上网用户每周访问网上资源平均4小时。
　　6．在“Size”框中输入Java Applet运行时屏幕上所占的高度和宽度。目前，大多数用户在网上查询信息是通过搜索引擎，常用的搜索引擎不过几个，这就说明了存在大量的重复访问相同的站点。
　　8．在“Alignment”框中设置Java Applet与同一行文本之间的对齐方式。
　　全部设置完毕，按“OK”即完成了对Java Applet程序的插入工作。
