框架结构模态分析的行波方法*
朱桂东　郑钢铁　邵成勋
　　摘　要　基于空间框架结构的行波动力学模型，给出获取结构模态参数的一个新方法。将由一维波导组装成的框架结构的振动作为弹性波研究，弹性波在波导上传播，在结点处反射和散射。结构的模态特性在结构呈自由振动状态时得以体现，据此建立一个关于结构内行波的特征值问题，并由此求得结构的固有频率及振型。文中还给出了与有限元方法结果的比较
　　主题词　框架结构　模态分析　行波
MODAL ANALYSIS FOR FRAME STRUCTURES 
VIA TRAVELING WAVE APPROACH
Zhu Guidong
(Beijing Institute of Astronautical System Engineering.Beijing.100076)
Zhang Gangtie Shao Chengxun

　　Abstract　Based on traveling wave formulation,a new approach to obtain modal parameters for frame structures is presented.A frame structure is composed of one-dimensional members,and the vibration of the frame is regarded as travelling waves propagating along the members.From free vibration model of a frame structure,an eigenvalue problem is formed for wave mode vector.Natural frequencies and corresponding modal shapes for the structure are obtained by solving the eignvalue problem.Comparison with the finite element method is also presented.
　　Key words　Frame structures Modal analysis Travelling wave
　　1　引言
　　结构模态参数的提取在结构振动分析中具有相当重要的位置。
　　.制定合理的操作工艺根据以上各项的要求，制定合理的镀膜工艺，镀出的轴承外圆增厚膜层经测试性能如下：膜层厚度按要求可控制在1～5μm，显微硬度（0.2kg）在742～793N/mm2之间，大体与基体硬度相吻合，外圆精度也完全能满足使用要求。对运动方程及边界条件都很简单的结构，如均质简支梁、悬臂梁等，其模态特性可从一函数特征值问题解得［2］。
　　除上述几个应用实例外，多弧离子镀技术在航空测试用仪器的主轴、航空仪表、各种零件等方面也都有使用。
　　由一维单元组装成的框架结构是现代大型空间结构的一种主要形式，尽管整个结构的构型可能非常复杂，但每个单元都是很简单，其模型一般可精确地建立。该膜层应具有耐高温、耐烧蚀、与叶片基体结合牢固、抗冷热气流冲击及结构致密、厚度均匀的特点。Beale等［4］用行波分析了结构中的功率流，并给出能量传递路径。MacMartin等［5］进行了基于行波模型的振动控制工作。朱桂东等［6，7］基于行波模型对框架结构的阻尼减振进行了分析。王泉等［8］讨论了结构波动区域的可控性及降维问题。
　　尽管行波模型已被广泛地研究，但波动模型与传统的模态空间模型的关系却很少被研究。Meirovitch［9］曾研究在模态空间对行波进行控制。本文研究从行波模型中获得框架结构的模态特性，如固有频率、模态振型等。同时给出了与有限元方法结果的比较。
　　2　框架结构行波动力学模型［6，7］
　　框架结构由一维的单元(图1)通过结点(图2)组装而，在每一个单元上有不同频率的拉压、扭转、弯曲波传播。对每一个单元的要求是其必须具有连续的机械阻抗，而且所有的外力和位移边界条件都作用在结点上，因此波在单元上传播时无反射。称每一种独立传播的波得傅利叶变换为波模式(Wave Mode)。单元任一截面上的波模式向量定义为wm=［wTl,wTr］T，其中wl和wr分别称为左行波模式向量和右行波模式向量。在其表面镀制了2～5μm的TiN膜层后，其表面显微硬度可达HV2700以上，经精研后表面光洁度≤Ra0.033时，表面显微硬度仍能达到HV2100以上。为了提高涡轮叶片的寿命，增强其抗高温烧蚀的能力，需在涡轮叶片表面镀制一层热障膜层。Wi和Wo分别称为系统出射和入射波模式向量。

　　3　模态分析
　　结构的模态特性可从其自由振动中反映出来，从自由振动的动力学模型求得位移的非零解即为结构振动的模态位移。令A(ω)=T-S,A是n×n维复矩阵，n为入射波模式的维数。通过几年的推广，已在航空航天工业的多类零部件如轴承、摄影器件、机加工刀具、涡轮叶片上应用，效果令人满意。令(2)式中的外激励向量R为零，结构自由振动的动力学方程为
　　　　　　　　　　　　　　　　　　　A(ω)Wi=0　　　　　　　　　　　　　　　　　(3)
方程(3)是一个特殊的特征值问题，矩阵A的元素为频率ω的超越函数。结构当且仅当方程(3)具有非平凡解时才能维持自由振动状态，即
　　　　　　　　　　　　　　　　　　　det(A)=0　　　　　　　　　　　　　　　　　(4)
上式是一个超越方程，可以看成是结构的特征方程式。本文仅考虑无阻尼的情况，因此(4)式的无穷多个解ωi,1=1,1,…均为实数，这些解对应着结构的无穷多个固有频率。
　　我国第一代特殊用途卫星测量照相机的镜片托架、镜筒、支撑框架、焦面框架等件采用的都是钛合金材料，其表面处理问题一直无法解决，若采用氧化发黑工艺处理，膜层发亮发灰，且容易脱落，也不符合光学系统消除杂散光的要求；若用无光黑漆处理，则系统真空干燥后漆层掉屑将影响系统使用。

式中ri为A(ωi)的缺秩数。令{α1,α2,…,αri)为A(ωi)零空间的一组基，因此有

　　由于上述方程的复杂性，对略复杂的结构，其结果只能够用数值的方法求出。求解过程中的难点在于解特征方程式(4)，以获得结构的固有频率。不适当的搜索方法可能会遗漏某些固有频率，因此在求解过程中应利用结构的各种频率响应，以判定固有频率的位置及迭代初值。

　　本文基于行波模型给出获得框架结构的模态特性的一种新方法。
结论
　　多弧离子镀工艺是一项新的表面处理技术，由于离子镀膜层所具有的优点越来越受到人们的重视。动压气浮陀螺马达球轴承的球及碗的表面采用多弧离子镀工艺处理后，可增加球及碗表面的硬度，提高耐磨性。用多弧离子镀膜工艺镀制NiCrAlY热障涂层已成功地应用于航空发动机涡轮叶片的表面处理上，经试验及实际应用，取得了满意效果，并逐步应用到多种型号发动机的涡轮叶片的表面处理上。行波模型给出了非常全面的结构特性，但同时其计算量也比较大，如何使之成为一种工程实用的方法还需要很多的研究。
