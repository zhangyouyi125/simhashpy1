Intelligent Ultrasonic Mass Level Remote Measuring System
Jing Bo
　　Abstract　The principle of an intelligent ultrasonic measuring system for mass level with remote control is presented.This system has feartures of high accuracy of measurement,high speed (fast response) and small repeatability error, and has the function of data access memory.
　　Key words　Ultrasonic transducer,Mass level measurement,Remote control
　　超声波测位技术是近年来涌现出来的许多测位新技术之一，由于它具有不接触被测介质、结构简单、可靠性高等优异特性，获得了越来越广泛的应用。但是，目前常见的测位系统功能单一，精度不高，不能进行远距离实时监测，因此，在我们研制的料位测量系统中，不仅采用单片机为控制核心实现智能监测，而且将编码通信技术应用于本系统，实现了远距离实时监测。是否就可以按照现有的标准(国标或国军标)来进行相应的试验了呢。利用声在媒质中的声速、衰减、共振、反射等现象可测量物质的成分、比重、厚度等。遗憾的是综合箱一般并没有给出设备的风速到底有多大，很容易造成人们的错觉，误认为综合箱还可以做其它的试验。本系统原理框图见图1和图2。遗憾的是综合箱一般并没有给出设备的风速到底有多大，很容易造成人们的错觉，误认为综合箱还可以做其它的试验。
　　至于其它标准对试验箱风速的要求，在标准中已给出了说明，是为防止试验样品内产生不符合实际的(冷或热)传导，试验目地是考验产品在各种温度条件下贮存和工作的适应性\.通常，这种试验在某种温度情况下试验时间较长，风速既有模拟环境的作用又有均匀试验箱的作用，当达到某种试验温度时，要求保温2 h，使试验箱内温度缓慢地达到均匀，可见它与可靠性试验有很大的不同。其中，H为贮罐高度。

2　系统特点
　　1)通用性好。采用超声波专用集成电路LM1812，不仅外围元件较少，电路简单，尺寸小，而且有更好的稳定性及可靠性；单片机控制，实现对信号的收发控制、判读、温度补偿和数据转换处理等功能，集成度高。
　　2)功能强。系统有自动测量功能,温补功能,上下限告警功能；采用编码通信技术，实现无线遥控。
　　从分析中我们知道，风速的不同，导致这两类不同的设备试验目地不同，当然，我们总希望设备能符合的标准越多越好，实际上是有困难的，比如现有的综合箱风速是不可调的，如改成可调，并调到符合GJB150的要求，那么在多大温度范围内能符合温度误差，恐怕就很小了，这样就失去了意义；所以，在实际工作中，应该对试验设备的基本的技术指标有一个比较清楚的了解，特别是综合箱，它真正的用途就是用于产品的可靠性试验，而不适于做其它的试验，如果将它用于其它的试验，会使产品得不到真正的考验，产生不良的结果。因为在GJB899标准里，试验剖面里的温度变化速率与飞行高度和马赫数变化有关，它包括了产品在使用中可能遇到的各种情况，试验剖面实际上反应了试验件在不同的飞行高度和马赫数时的温度，而这种不同的飞行高度和马赫数时的温度是在不断变化的，与其它试验相比，这种试验就是动态试验，对产品的主要影响是各种温度和温度变化速率，与试验箱内风速的大小没有关系，同时，在制定温度时，考虑到进入机舱空气气流和进入机舱气流温度；因此，该标准主要要求的是能够提供一种温度、湿度、振动综合试验的环境，并能满足温度误差与温度变化速率的设备，以便给出产品合理的试验寿命，综合箱正是满足这种要求的设备，它能按照试验剖面的要求进行连续的试验。

3.2　软件设计
　　单片机控制超声波的发射与接收，收发装置将发射波与回波之间的时间间隔用脉冲宽度表示出来，外部计数器对其进行计数，单片机取得计数值后，对数据处理计算，得出距离值，驱动显示器显示。由于声速与传播介质的温度变化有很大关系，因此还需测量周围环境的温度，修正声速，从而提高精度。软件流程图如图3所示。
